# Intelligent Hybrid AI Framework for Real-Time Cyber Threat Detection

## 🔍 Overview
This project proposes a Hybrid AI-Based Intrusion Detection System (IDS) integrating:
- Random Forest
- Isolation Forest
- Support Vector Machine (SVM)

The system detects cyber threats in real-time using machine learning and anomaly detection techniques.

## 📊 Dataset
CICIDS 2017 – Real-world network traffic dataset.

## 🧠 Methodology
1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Hybrid Model Training
5. Threat Classification
6. Alert Generation

## 📈 Results
Accuracy: 98.4%  
F1 Score: 97.6%

## 🚀 Key Innovations
- Hybrid AI + Anomaly Detection
- Reduced False Positives
- Real-Time Classification
- Scalable for IoT & Cloud
